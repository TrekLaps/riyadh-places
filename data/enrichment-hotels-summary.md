## Enrichment Fields Added

For each hotel/chalet, the following fields were added:

- `rating` — Google Maps rating (1-5)
- `rating_count` — Number of reviews
- `phone` — Saudi format (+966...)
- `hours` — Operating hours in Arabic
- `website` — Official URL
- `instagram` — Instagram handle
- `description_ar` — Arabic description (real, not generated)
- `perfect_for` — Arabic tags (عائلات, شهر عسل, etc.)
- `price_level` — 1-4 (1=budget, 4=luxury)
- `amenities` — Arabic amenity tags (مسبح, واي فاي, etc.)
- `booking_url` — Booking.com or similar link
- `address_ar` — Full Arabic address
- `neighborhood` — Neighborhood name
- `check_in` / `check_out` — Check-in/out times

## Data Sources

- Google Maps ratings and review counts
- Official hotel websites (Marriott, Hilton, IHG, Accor, Hyatt, etc.)
- Travel Weekly GDS data
- Booking.com listings
- Instagram verified handles
- Saudi tourism directories
- Gathern.co for chalet bookings

## Notes

1. **Only REAL data used** — fields set to `null` when unverifiable
2. Price levels converted from $ notation to numeric 1-4 scale
3. Default check-in/out times set based on industry standards (hotels: 14:00/12:00, chalets: 16:00/14:00)
4. Booking URLs auto-generated for hotels without specific verified links
5. Many chalets are listed on platforms like Gathern, Estraha.com rather than having standalone websites
6. Some newer/upcoming hotels (Rosewood, Aman, One&Only, etc.) may have limited verified data as they haven't opened yet
